# Security Header Analyzer

A simple GUI tool that checks whether a website is using important security headers.

## Clone the Project

git clone https://github.com/mars13-tech/Security-Header-Analyzer.git 

cd Security-Header-Analyzer

## Install Requirement

pip install requests

## Run the Project

python main.py

## How to Use
1. Enter a website URL (example: google.com)
2. Click **Scan**
3. The tool will show which headers are:
   - ✅ Present
   - ❌ Missing
